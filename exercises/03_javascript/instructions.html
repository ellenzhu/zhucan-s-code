2. Javascript Arrays

In this exercise, you should implement three functions in JavaScript. This is in order for you to get familiar with working with arrays and objects in JavaScript.

The functions to implement

keywordusage(text, keywords) [5 points]

The function keywordusage returns which of the keywords are present in the text. Whitespace is used to separate words in the text and punctuations are part of the words. For example, call

keywordusage('Dive Into Python is a free book for experienced programmers', ['Python', 'python', 'scala'])

should return an array: [true, false, false].

frequencies(text, wordlist) [5 points]

The function frequencies should return an object with properties for each of the word in the wordlist. The values of the properties should be how many times the word is present in text. For example, call

frequencies('foo foo bar foo   bar buz', ['foo', 'bar'])

should return {"bar": 2, "foo": 3}.

rotate(array, steps)

Rotate all elements in the array. Second argument of the function is used to define how many steps the list is rotated. For example, if steps had the value 1, then all the list items would be moved to the right from their current position by one position except the last item, which would be moved to the first position of the list and thus we say that we rotate the list.

If steps is negative, the items are rotated to left instead. The steps can also be more than the size of the array, in which case the items rotate over the array more than once. For example, for an array with four elements, rotating by 1 would have the same result as rotating by 5.

The function should modify the existing array instead of creating a new one. In addition, it should return that same array.

Some examples:

var a = [1, 2, 3, 4];
rotate(a); // returns [4, 1, 2, 3]
rotate([4, 1, 2, 3], -1); // changes a to [1, 2, 3, 4] and returns it
rotate([1, 2, 3, 4], 10); // changes a to [3, 4, 1, 2] and returns it
rotate([3, 4, 1, 2], -5); // changes a to [4, 1, 2, 3] and returns it
rotate([4, 1, 2, 3], 0);  // changes a to [4, 1, 2, 3] and returns it

3. JavaScript Objects

In this exercise you are to implement a JavaScript function called addPersonMethods that takes and object (that has name and age properties defined) and returns an object that has those same properties and additional methods .greet(greetingString), .compareAge(anotherPersonObject) and .namesake(anotherPersonObject).

The following example illustrates how the function should work and what the defined methods should do:

var bob_def =  { name: 'Bob', age: 21 };
var eve_def =  { name: 'Eve', age: 21 };

var eve = addPersonMethods(eve_def);
var bob = addPersonMethods(bob_def);
var another_bob = addPersonMethods({name:'Bob', age: 40});

eve.greet("Hi all");
>>> Hi all, my name is Eve

eve.compareAge(bob);
>>> My name is Eve and I'm equally young as Bob

bob.greet("Hello");
>>> Hello, my name is Bob

eve.namesake(bob);
>>> We have different names, Bob and I.

another_bob.compareAge(eve);
>>> My name is Bob and I'm older than Eve

eve.compareAge(another_bob);
>>> My name is Eve and I'm younger than Bob

another_bob.namesake(bob);
>>> We have the same name, Bob and I!

Use console.log for output which is denoted in the example with >>>

4. jQuery Link Decoration

On many sites, you see little icons next to links that indicate, for example, the type of the linked file. In this exercise, you need to write code that will do this. Your code should:

  • be executed when the DOM is ready
  • add CSS class pdf to a link where the target ends with the suffix .pdf
  • add CSS class download to a link where the target ends with any suffix (.*) and does NOT end with .pdf or .html

Where to start

Open the exercises/03_javascript/linkdecoration.html in browser. Note, that the grading is done using the original HTML file. Do do not make your answer depend on changes to the HTML. The HTML template loads jQuery 1.11.1 and the exercises/03_javascript/linkdecoration.js file where you write your code.

  1. Try first to select all links and do something to them to see how things work.
  2. Now try to target the pdf-links only. (hint: look in the learning material links below - there might have been some useful things in the questionnaire as well)

Hints

  • $(this) creates a jQuery object from this DOM element. Might be handy when using the filter function.
  • Check out the jQuery selectors.
  • Please do not use the href-attribute of DOM nodes like this: this.href - use the JQuery object instead $(this).attr("href").
